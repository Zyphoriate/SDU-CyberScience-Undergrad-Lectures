1.1 μC/OS-II 简介
μC/OS-II，是⼀个专⻔为嵌⼊式应⽤设计的实时操作系统。除了⼀些和处理器相
关的代码⽤汇编编写，其他绝⼤部分代码使⽤ C 语⾔，所以在学习了 C 语⾔程序设计
和计算机组成原理，阅读这个操作系统的代码并不困难。该操作系统具有很好的可扩
展性，⼈们可以在它具有很好的可扩展性，⼈们可以在这个系统的基础上⼆次开发很
多功能。同时这个系统是开放源码的（不开放源码也很难有机会去学习）
，但是并不代
表这个操作系统是免费的（当然对于学习⽬的是免费）
，如果使⽤这个操作系统去完成
某个产品，那么就需要购买昂贵的版权许可证。当然这也从侧⾯说明了这个操作系统
的价值：既然⽤作商⽤，⾄少说明它是⼀个相当“靠谱”的系统。另外，该系统也被⼴泛
使⽤在从照相机到航空电⼦产品的各种应⽤中。

本实验选择 ARM Cortex-M3 的嵌⼊式微处理器作为硬件环境的处理器，主要是出于以
